# analyzing-grammy-data

#Overivew
Web analytics case study analyzing traffic, engagement metrics, and audience demographics for Grammys.com following the 2022 website split.

## Methods
- Cleaned and explored web analytics data using Python and Pandas
- Performed exploratory data analysis (EDA)
- Analyzed sessions, page views, traffic trends, and event-driven spikes
- Identified anomalies and high-impact periods

## Key Insights
- Website traffic increased significantly around major Grammy events
- User engagement patterns were strongly tied to announcements and show dates
- Data-driven recommendations were developed to improve year-round engagement

## Tools
Python, Pandas, Data Visualization
